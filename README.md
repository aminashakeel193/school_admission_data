# school_admission_data
Data management, PHP, MySQL,  
School Admission Data System

A web-based School Admission Management System built using PHP, MySQL, HTML, and CSS.

##  Features
- **Student Registration**: Online admission form for new students
- **Admin Dashboard**: View, edit and delete student records
- **Login System**: Secure admin login with session management
- **Database Storage**: All data stored in MySQL
- **Contact Form**: Messages saved directly to database
- **Responsive Design**: Works on mobile and desktop
- **CRUD Operations**: Create, Read, Update, Delete student data

##  Technologies Used
- **Frontend**: HTML, CSS
- **Backend**: PHP
- **Database**: MySQL
- **Server**: XAMPP

##  How to Run
1. Import `database.sql` file in phpMyAdmin
2. Rename `config_sample.php` to `config.php` and add your database credentials
3. Run the project on localhost using XAMPP

## Purpose
This project demonstrates full-stack web development skills and ability to handle large datasets for educational institutions.
